# PathFinderIT

A web-based career interest assessment tool for computer science students. Helps you discover which IT field aligns best with your interests through a guided questionnaire.

## What is This?

PathFinderIT is an interactive quiz application that analyzes your preferences across four major computer science domains:

- **Artificial Intelligence** - Machine learning, neural networks, automation
- **Software Engineering** - Application development, system design, coding practices
- **Data Science** - Analytics, statistics, data-driven insights
- **Computer Networks** - Infrastructure, security, connectivity

Based on your responses, the app calculates a ranking and provides personalized insights to help guide your career path.

## How It Works

1. **Take the Quiz** - Answer 12 questions about your interests (1-5 scale)
2. **Get Scored** - The SMART method calculates your preference scores for each domain
3. **View Results** - See your ranking with a visual chart and detailed analysis
4. **Get Insights** - Read personalized recommendations based on your profile

## Features

- **Responsive Design** - Works smoothly on desktop, tablet, and mobile
- **Admin Dashboard** - Track user responses and export data
- **Dynamic Analysis** - 80+ text variations ensure unique insights each time
- **Score Pattern Detection** - Identifies if you have clear specialization or broad interests
- **Persistent Storage** - Save and review results from the database

## Tech Stack

- **Frontend**: Bootstrap 5.3.0, Chart.js, Vanilla JavaScript
- **Backend**: PHP 7.x
- **Database**: MySQL
- **Server**: Apache (Laragon recommended for development)

## Installation

### Requirements

- PHP 7.0 or higher
- MySQL 5.7 or higher
- A web server (Apache, Nginx, etc.)

### Quick Start

1. Clone or download the repository
2. Import the database structure (if provided)
3. Update database credentials in `config/connection.php`
4. Place files in your web root
5. Access via `http://localhost/pathfinderit`

### Database Setup

Create a database and configure the connection:

```php
// config/connection.php
$servername = "localhost";
$username = "root";
$password = "";
$database = "pathfinderit";
```

## The SMART Scoring Method

Each domain's score combines 3 key questions:

- **AI**: Q1, Q9, Q12(cost weight) → 0.4, 0.4, 0.2
- **SE**: Q2, Q4, Q11 → equal weights
- **DS**: Q3, Q7, Q10 → equal weights
- **CN**: Q5, Q6, Q8 → equal weights

Final scores normalize to 0-1 range.

## Note on Results

Results are recommendations, not career determinants. They're best used as a starting point for exploring your interests further. Actual career fit depends on experience, continuous learning, and personal passion.
